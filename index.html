HTML

  • The head element contains information about the webpage
  • The body element represents the the visible content to the user
  • The meta element contains information about the page that is used by the browser
  • Elements that are nested inside other elements are called child elements
  • An element refers to the entire element from opening tag to closing tag, whereas a tag refers only to what appears inside the angle brackets

CSS

  • The most common way to style a webpage is by using an external CSS style sheet
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it

JavaScript

  • Control flow is the order in which a computer executes code in a script.
  • In JavaScript, code runs from the first statement at the top of the page, down to the last statement at the bottom of the page
  • A value is considered falsy if it is one of the following values: 0, -0, On, "", null, undefined, NaN, or a Boolean data type with the value of false
  • Using each piece of data's index, or location number, we can access individual data inside the array
  • A function is simply a set of instructions that tells the computer how to perform a certain task
  • A variable is a named container that allows us to store data in our code.
